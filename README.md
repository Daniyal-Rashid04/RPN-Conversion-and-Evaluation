# RPN-Conversion-and-Evaluation
Reverse Polish Notation (RPN) Calculator – DSA Project

This project is a Reverse Polish Notation (RPN) Calculator developed using Data Structures and Algorithms (DSA) concepts with a simple graphical user interface.
Features:
Stack-based expression evaluation
User-friendly interface
Supports basic arithmetic operations:
-Addition
-Subtraction
-Multiplication
-Division
Error handling for invalid expressions
Demonstrates practical implementation of stacks in DSA
